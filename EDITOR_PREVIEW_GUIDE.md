# SDS编辑器预览功能使用指南

## 概述

SDS编辑器提供了强大的预览和编辑功能，让用户可以在编辑文档的同时实时查看最终渲染效果。

## 功能特性

### 🎯 预览功能
- **实时预览**：编辑内容实时同步到预览区域
- **SDS语义块渲染**：支持所有语义块类型的正确渲染
- **表单定义预览**：显示交互式表单的最终效果
- **响应式布局**：适配不同屏幕尺寸
- **完整样式支持**：包括代码高亮、表格样式等

### ✏️ 编辑功能
- **富文本编辑**：支持粗体、斜体、删除线、代码等格式
- **Markdown语法**：支持标准Markdown语法
- **工具栏操作**：提供便捷的格式化按钮
- **撤销/重做**：支持编辑历史操作
- **语义块插入**：通过侧边栏插入语义块
- **表单定义插入**：创建交互式表单

## 使用方法

### 1. 基本操作

#### 切换到预览模式
1. 在编辑器工具栏右侧找到"预览"按钮（👁️图标）
2. 点击"预览"按钮
3. 编辑器会切换到预览模式，右侧显示渲染后的文档
4. 预览内容会实时更新

#### 切换回编辑模式
1. 在预览模式下，工具栏右侧的按钮会显示"编辑"
2. 点击"编辑"按钮
3. 编辑器会切换回编辑模式，隐藏预览面板

### 2. 工具栏功能

#### 格式化按钮
- **粗体** (B)：选中文本后点击，或使用快捷键 `Ctrl+B`
- **斜体** (I)：选中文本后点击，或使用快捷键 `Ctrl+I`
- **删除线** (S)：选中文本后点击，或使用快捷键 `Ctrl+Shift+S`
- **代码** (`)：选中文本后点击，或使用快捷键 `Ctrl+``

#### 列表按钮
- **无序列表** (•)：创建项目符号列表
- **有序列表** (1.)：创建编号列表
- **嵌套列表**：在列表项中按Tab键缩进

#### 表格按钮
- **插入表格**：点击表格按钮插入3x3表格
- **调整列宽**：拖拽表格列边界
- **添加行列**：在表格中右键选择操作

#### 其他功能
- **撤销** (↶)：撤销上一步操作，快捷键 `Ctrl+Z`
- **重做** (↷)：重做上一步操作，快捷键 `Ctrl+Y`
- **保存**：保存当前文档内容

### 3. 侧边栏功能

#### 语义块编辑器
1. 点击工具栏左侧的设置按钮显示侧边栏
2. 选择"语义块"标签
3. 选择语义块类型（信息、警告、危险、提示、注意）
4. 填写标题和内容
5. 点击"确认"插入语义块

#### 表单定义编辑器
1. 在侧边栏中选择"表单定义"标签
2. 点击"插入表单"按钮
3. 配置表单基本信息（ID、标题、描述）
4. 添加表单字段（文本、邮箱、数字、选择等）
5. 设置验证规则
6. 点击"确认"插入表单定义

## 预览内容说明

### 支持的格式
- **标题**：H1-H6标题层级
- **段落**：普通文本段落
- **列表**：有序和无序列表
- **代码**：行内代码和代码块
- **表格**：数据表格
- **引用**：引用块
- **语义块**：信息、警告、危险、提示、注意块
- **表单**：交互式表单

### 语义块样式
- **信息块** (info)：蓝色背景，用于重要信息
- **警告块** (warning)：黄色背景，用于提醒注意
- **危险块** (danger)：红色背景，用于警告风险
- **提示块** (tip)：绿色背景，用于有用建议
- **注意块** (note)：橙色背景，用于强调内容

## 常见问题

### Q: 预览内容不显示怎么办？
A: 
1. 确保编辑器中有内容
2. 点击"预览"按钮切换到预览模式
3. 检查浏览器控制台是否有错误信息
4. 刷新页面重试

### Q: 预览内容不更新怎么办？
A:
1. 确保在预览模式下编辑内容
2. 预览内容会实时同步更新
3. 如果仍有问题，可以重新点击"预览"按钮

### Q: 语义块在预览中显示不正确怎么办？
A:
1. 检查语义块语法是否正确
2. 确保使用了正确的语义块类型
3. 查看浏览器控制台的错误信息

### Q: 工具栏按钮不响应怎么办？
A:
1. 确保编辑器已加载完成
2. 选中要格式化的文本
3. 检查浏览器控制台是否有错误
4. 刷新页面重试

## 技术说明

### 预览实现原理
1. 编辑器内容以HTML格式存储
2. 将HTML转换为Markdown格式
3. 使用SDS解析器解析Markdown
4. 渲染为最终HTML显示

### 支持的格式转换
- HTML ↔ Markdown 双向转换
- SDS语义块解析和渲染
- 表单定义解析和渲染
- 代码语法高亮
